# BSM-SIM: 3D Corporate IT Simulation (TÜBİTAK 2209-A)

**Status:** Active Development  
**Technologies:** Unity 3D, C#  
**Domain:** Serious Games, Experiential Learning, Information Systems Engineering

## About the Project
* A 3D FPS serious game is being developed under the TÜBİTAK 2209-A and Senior Graduation Project, transforming the Information Systems Engineering curriculum into an experiential learning model through corporate scenarios.
* Various corporate business problems are gamified into an expandable problem set using a modular architecture and interactive in-game interfaces.
* The project aims to deliver a demo (MVP) for the graduation thesis, while the TÜBİTAK phase focuses on measuring and reporting the final product's impact on students.
